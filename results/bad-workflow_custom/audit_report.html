F5 Untrusted input may reach 1 AI node(s): Node1
A graph path exists from a trigger node to an LLM node. Anything a caller sends can influence the prompt (OWASP LLM01:2025).
Remediation:
  1. Sanitize and strictly validate all user-controlled fields before they reach the LLM (allowlist, length cap, type checks).
  2. Use a system prompt that explicitly refuses to follow instructions found inside user data.
  3. Add an output guardrail step (regex / classifier / second LLM judge) before any side-effecting action.
  4. Where possible, place untrusted text inside a clearly delimited block and instruct the model to treat it as data only.
